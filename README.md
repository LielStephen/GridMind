# GridMind

GridMind is a reinforcement learning grid-world simulator and training platform.

## Directory Structure

```
GridMind/
│
├── backend/            # Python backend code
│   ├── api/            # API endpoints & server setup
│   ├── rl/             # Reinforcement Learning algorithms
│   ├── simulator/      # Grid-world environment simulation logic
│   ├── models/         # Neural network architectures or saved models
│   ├── config/         # App and training configuration files
│   └── utils/          # Auxiliary scripts, helpers, and pathfinders
│
├── frontend/           # UI visualization panel and editor
│
├── datasets/           # Environment layouts and generated transition logs
│
├── docs/               # Technical design and user documentation
│
├── tests/              # Automated tests
│
├── training/           # Standalone scripts for offline agent training
│
├── docker/             # Containerization configurations
│
├── README.md           # Project documentation
│
├── requirements.txt    # Python dependencies list
│
└── .gitignore          # Git exclusion patterns
```
